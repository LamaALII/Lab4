# Lab4
This lab focuses on assessing and improving the quality of a student performance dataset through data preprocessing techniques. The main objective is to identify potential data quality issues and prepare the dataset for further analysis.

The process includes handling missing values using an appropriate strategy, detecting and removing outliers using the IQR method, and normalizing numerical features using both Min-Max scaling and Z-score standardization. Additionally, correlation between numerical features is examined to determine whether dimensionality reduction using PCA is necessary.

Overall, this lab demonstrates essential data preprocessing steps that enhance data reliability and ensure it is suitable for machine learning and analysis tasks.
